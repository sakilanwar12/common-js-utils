## multiply(a, b) ⇒ number

Calculates the product of two numbers.

## Parameters

| Name | Type     | Description        |
| ---- | -------- | ------------------ |
| `a`  | `number` | The first number.  |
| `b`  | `number` | The second number. |



## Returns

`number` — The result of multiplying `a` by `b`

## Example

```
multiply(2, 3);      // => 6
multiply(-4, 5);     // => -20
multiply(0, 10);     // => 0
multiply(3.5, 2);    // => 7


```
